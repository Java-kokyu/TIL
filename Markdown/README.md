# Markdown 문법

## Emphasis(강조)
기울임꼴 italic  
*hello world*

굵게 bold  
**hello world**

취소선 strikethrough  
~~hello world~~

기울림꼴 + 굵게  
***hello world***

굵게 + 취소선  
**~~hello world~~**  
~~**hello world**~~
